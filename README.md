# 🛡 CyberQuest AI

CyberQuest AI is an AI-powered cybersecurity escape room game where players solve incident response challenges generated by Large Language Models (LLMs).

## 🚀 Features

* User Authentication (Signup/Login/Logout)
* Room Creation and Join System
* AI-Generated Cybersecurity Challenges
* Dynamic Hint System
* Answer Validation
* Mission Timer
* Score Tracking
* Victory Screen
* Responsive Modern UI

## 🏗 Architecture

### Frontend

* Next.js
* React
* TypeScript
* Tailwind CSS

### Backend

* Next.js API Routes
* Node.js

### Database

* SQLite

### AI Integration

* Groq API
* Llama 3.3 70B Versatile

## 🎮 How It Works

1. User signs up and logs in.
2. Create a mission room.
3. AI generates a cybersecurity challenge.
4. Player solves the challenge.
5. Request hints if needed.
6. Submit the answer.
7. Complete the mission and achieve victory.

## ⚙️ Installation

Clone the repository:

```bash
git clone <repository-url>
cd cyberquest-ai
```

Install dependencies:

```bash
npm install
```

Create `.env.local`:

```env
GROQ_API_KEY=your_api_key
```

Run the project:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
